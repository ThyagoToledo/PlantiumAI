## Objetivo

Criar uma nota atômica bem-formada em `concepts/`.

## Pré-requisitos

- Informação pronta (pesquisada ou existente)
- Entendimento claro do conceito

## Passos

1. Abra `TEMPLATE_CONCEPT.md`
2. Copie para `concepts/[nome-topico].md`
3. Preencha: Definição (1 frase), Contexto, Detalhes (bullets)
4. Linke no `00_MOC.md` na seção apropriada
5. Commit + push

## Validação

- Arquivo tem apenas 1 ideia principal?
- Definição é clara em uma frase?
- Sem prosa, apenas bullets e código?
- Links apontam para arquivos que existem?

## Troubleshooting

- Nota muito longa? → Divide em 2 notas
- Muitas dependências? → Cria notas pré-requisito primeiro
