## Resumo

Documentação de modelos, pricing e capacidades da Claude API por Anthropic.

## Modelos Atuais (Feb 2025)

- Claude 3.5 Sonnet: Melhor custo/performance, recomendado para maioria dos casos
- Claude 3 Opus: Mais poderoso, custo mais alto
- Claude 3 Haiku: Mais rápido e barato, para tarefas simples

## Pricing (tokens)

- Input: $0.003/1k (Sonnet), $0.015/1k (Opus)
- Output: $0.015/1k (Sonnet), $0.075/1k (Opus)

## Features Importantes

- Token counting SDK disponível (Python/JS)
- Batch API para processar em volume
- Vision capabilities (análise de imagens)
- Tool use (function calling)
